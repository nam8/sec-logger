# SEC Service Logger

A mobile and web prototype app for clergy in the Scottish Episcopal Church to log sacramental events and generate the Canon 50 Annual Statistical Return.

Log each Holy Eucharist, Reserved Sacrament communion, Baptism, Confirmation, Wedding, or Funeral in a few taps. The app computes Easter and Advent dates automatically, tracks attendance and communicant totals, and keeps a running tally of the annual statistical return fields (§3–§14). 

**Disclaimer:** This is a freelance project by a private developer. The Scottish Episcopal Church is not formally affiliated with it in any way. 

**Stack:** React, Vite, Supabase (auth + Postgres), GitHub Pages. No server to maintain.
